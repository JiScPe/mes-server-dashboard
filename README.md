# mes-server-dashboard
 Modules status on every MES server

## Requirement Tools
1. Node.js [download-link](`https://nodejs.org/en/download/current`)
2. Git [download-link](`https://git-scm.com/install/`)
3. GithubDesktop(Optional) [download-link](`https://desktop.github.com/download/`)

## Get Started
Check all tools are installed by run these commands.

`node -v`

`npm -v`

`git -v`

## Clone project from Github repository
### using Git command
create project folder and navigate to that folder in ternimal

- in terminal type `git clone https://github.com/JiScPe/mes-server-dashboard.git`
-  `cd mes-server-dashboard`
- `npm install`
- duplicate .env.example file
- change file name .env.example  into .env ~~.example~~
- run `npm run dev`
- Go to http://localhost:3000 for test
